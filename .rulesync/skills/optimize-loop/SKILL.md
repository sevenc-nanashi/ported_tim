# optimize-loop

AviUtl2 MCPを使って、自分自身でコードを最適化する方法を説明します。

まず、AviUtl2 MCPを使い、パラメーターを変えて最適化前の結果を保存しておきます。
次に、コードを最適化します（optimize Skillを参考にしてください）。
最後に、最適化後のコードをAviUtl2 MCPに適用し、同じパラメーターで結果を保存します。
最適化前と最適化後の結果を比較し、計算式が正しく最適化されているかを確認します。
このプロセスを繰り返すことで、最適化を人に頼らず自分で行うことができます。
